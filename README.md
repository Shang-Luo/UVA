# 图演优化 - 基于图论算法的无人机路径规划

## 项目背景

随着无人机技术的快速发展，多无人机协同工作系统在物流配送、灾害救援、农业监测等领域展现出巨大应用潜力。然而，复杂环境中的多无人机路径规划与协同避障仍然是一个具有挑战性的问题。

本项目是曼城联合学院第三届"UP训练营"的"翼起未来"无人机应用设计模块的核心内容。通过本项目，我们旨在：

1. **掌握软件开发协作工具**：学习Git版本控制系统和GitHub工作流，培养团队协作能力
2. **深入理解图论算法**：将理论算法应用于实际问题，通过时间/空间复杂度评估算法性能
3. **构建完整的无人机模拟系统**：从环境建模、路径规划到实时避障，实现完整的无人机模拟解决方案

## 项目目标

开发一个基于图论算法的多无人机路径规划与避障系统，实现：
- 单无人机在复杂障碍环境中的最优路径规划
- 多无人机协同工作，避免相互碰撞
- 实时雷达感知与动态避障
- 可视化模拟环境，支持算法性能评估

## 技术栈

### 核心编程语言
- **Python** - 主要开发语言，提供丰富的科学计算和算法库支持
- **C** - 算法开发语言，为大一新生提供语言支持

### 可视化与模拟
- **Pygame** - 2D图形渲染引擎，用于构建无人机和障碍物的可视化界面

### 算法实现
- **图论算法**：可见图算法、A*搜索算法、单元分解法、SPFA单源最短路径算法
- **几何计算**：射线-线段相交检测、多边形碰撞检测
- **物理模拟**：加速度-速度-位置集成，基于力的避障模型

### 开发工具
- **Git & GitHub** - 版本控制和团队协作
- **GitHub Desktop** - Git图形界面工具
- **VS Code** - 代码编辑器
- **JSON** - 配置文件和数据存储格式

## 项目架构

### 模块设计
```
图演优化系统/
├── 环境模拟层 (Pygame)
│   ├── 障碍物渲染
│   ├── 无人机可视化
│   ├── 雷达接口模拟
│   └── 实时状态显示
├── 算法核心层 (Python)
│   ├── 环境建模
│   │   ├── 障碍物网格化
│   │   ├── 可见图构建
│   │   └── 单元分解
│   ├── 路径规划
│   │   ├── 单源最短路径 (SPFA)
│   │   ├── A*搜索算法
│   │   └── 多目标优化
│   └── 避障控制
│       ├── 雷达数据处理
│       ├── 斥力场计算
│       └── 加速度控制
└── 评估分析层
    ├── 时间复杂度分析
    ├── 空间复杂度分析
    └── 总寻路时间统计
```

### 数据结构
- **障碍物**：多边形顶点列表，支持静态和动态障碍
- **无人机**：位置、速度、加速度、半径、雷达范围、状态标记
- **路径**：坐标点序列，支持平滑处理和动态更新
- **雷达数据**：方向性距离感知数组，用于实时避障决策

## 📈 性能评估指标

### 主要评估指标
1. **总寻路时间**：所有无人机从起点到目标的总时间
2. **时间复杂度**：算法执行时间随问题规模增长的趋势
3. **空间复杂度**：算法内存使用量随问题规模增长的趋势
4. **路径质量**：路径长度、平滑度、安全性评分

### 对比分析维度
- 不同算法在同一场景下的表现
- 不同场景复杂度下的算法稳定性
- 多无人机数量扩展时的性能变化
